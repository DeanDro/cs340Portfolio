Description of pages

  • Select the customer page to view all customers currently in database, to add or delete customers.
  • Select Rooms to view information about all Rooms in the database, to add or delete a room.
  • Select Reservations to view information about the Reservations currently in the database, to add or delete any Reservations.
  • Select Discounts to view information about the discount information currently in the database, to add or delete any discount offers.
  • Select Discounts - Reservations to view information about the discounts that are applied in each reservation, to add, delete or updated any of the information.
